System frame:

Timer:
    now(n):
	returns current time.
    stop(n):
	returns time span since last time to call stop(n), 0 if first time.
    rstop(n):
	returns time span since last time to call stop(n), 0 if first time.
    reach(n, t):
	returns whether time span is not less than last call of stop(n)/successful reach(n, t). 0 if first time.
    Re(n):
	reset nth timer.
    reset(n):
	reset nth timer.
    Fps(t):
	returns whether this time reaches frame time since last successful frame.
